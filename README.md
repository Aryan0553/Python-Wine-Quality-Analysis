# 🍷 Wine Quality Analysis Using Python

## 📌 Project Overview

This project performs an in-depth Exploratory Data Analysis (EDA) on the Wine Quality Dataset using Python. The objective is to understand the distribution of wine characteristics, identify patterns within chemical properties, and analyze factors that influence wine quality ratings.

The analysis includes statistical summaries, data visualization, distribution analysis, skewness detection, and feature interpretation to gain valuable insights into wine quality.

---

## 🎯 Objectives

- Understand the structure of the dataset.
- Perform data exploration and statistical analysis.
- Analyze the distribution of important wine attributes.
- Compare mean and median values of features.
- Identify skewness in numerical variables.
- Examine the quality score distribution.
- Generate meaningful visual insights using Python.

---

## 📊 Dataset Information

The dataset contains **4,898 wine samples** and **12 attributes**.

### Features

| Feature | Description |
|----------|-------------|
| Fixed Acidity | Non-volatile acids present in wine |
| Volatile Acidity | Acids contributing to vinegar taste |
| Citric Acid | Freshness and flavor component |
| Residual Sugar | Remaining sugar after fermentation |
| Chlorides | Salt content in wine |
| Free Sulfur Dioxide | Prevents microbial growth |
| Total Sulfur Dioxide | Total SO₂ concentration |
| Density | Density of wine |
| pH | Acidity level |
| Sulphates | Antimicrobial and antioxidant additive |
| Alcohol | Alcohol percentage |
| Quality | Wine quality score |

## 📌 Key Findings

### Fixed Acidity
- Average fixed acidity is approximately **6.85**
- Median value is **6.8**
- Distribution is relatively balanced

### Volatile Acidity
- Average volatile acidity is approximately **0.28**
- Positive skewness observed
- Most wines contain lower volatile acidity levels

### Citric Acid
- Average citric acid concentration is approximately **0.33**
- Median value is **0.32**
- Distribution indicates moderate variation

### Wine Quality
- Most wines received a quality score of **6**
- Quality ratings are concentrated between **5 and 7**
- Extremely low and high quality wines are rare
